# Command: git-sync\n\n**Description:**\nManually or automatically commit and push all changes to the current branch.\n\n**Usage:**\n```bash\nbash scripts/utilities/git-sync.sh\n```\n\n**How auto-sync works:**\n- Called by cron after daily/weekly checks\n- Uses latest changelog entry as commit message\n- Pushes to current branch\n\n**How to view CI status:**\n- Visit GitHub repo Actions tab\n- Or use:\n```bash\ncurl -s -H "Authorization: token $GITHUB_PAT_TOKEN" "https://api.github.com/repos/DrivenIdeaLab/agent-zero-evolution/actions/runs?per_page=1" | jq .workflow_runs[0] | grep -E "status|conclusion"\n```\n\n**Handling merge conflicts:**\n- Resolve locally, then rerun git-sync.sh\n
