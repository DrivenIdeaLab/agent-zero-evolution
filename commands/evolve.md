# Command: run evolution cycle

**What it does:**
Runs the full evolution cycle routine as defined in `scripts/utilities/evolution-cycle.md`.

**How to invoke:**
Say or type: `run evolution cycle`

**Procedure reference:**
- Checks for changes since last cycle
- Reviews evolution backlog
- Runs health check
- Proposes new backlog items
- Recommends top 3 actions
- Summarizes in a status report

**When to use:**
Periodically or after major changes.

**Expected output:**
Structured markdown report with all steps and recommendations.
