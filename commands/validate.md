# Command: validate registries

**What it does:**
Runs the registry validation script to check all registry markdown files for proper structure and completeness.

**How to invoke:**
Say or type: `run registry validation`

**Expected output:**
PASS/FAIL for each registry file, with error details if any fail.

**How to fix failures:**
- Remove placeholder/empty rows
- Ensure all required columns are filled
- Re-run until all PASS
