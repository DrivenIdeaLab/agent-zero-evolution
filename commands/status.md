# Command: system status

**What it does:**
Displays a concise dashboard of the current Agent Zero meta-project state, including configs, skills, hooks, prompts, docs, knowledge, validation, permissions, and commands.

**How to invoke:**
Say or type: `system status`

**Expected output format:**
Markdown table summarizing each area, recent changelog entries, and any warnings.
